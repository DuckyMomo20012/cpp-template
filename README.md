<!-- Run Locally -->

### :running: Run Locally

Clone the project:

```bash
git clone https://github.com/DuckyMomo20012/cpp-template.git
```

Go to the project directory:

```bash
cd cpp-template
```

Build program executable:

```bash
make
```

Run executable:

```bash
./main
```
